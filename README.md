# BG3-Item-Checklist
Checklist for getting all unique gear in Baldur's Gate 3. Up-to-date as of Patch 7.

**>> [View it here](https://joe1817.github.io/BG3-Item-Checklist/) <<**

## Included
- functionally unique gear
- mind flayer parasite specimens
- permanent bonuses

## Not Included
- common, +1, and +2 gear
- gear equivalent to above (e.g., Staff of the Moonmaiden, Magical Pike, most story gear)
- inaccessible gear
- items with no beneficial effect (e.g., Kill Collar)
- DLC items (e.g., Mask of the Shapeshifter)
